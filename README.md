# DesktopGui

---
## About
App with PyQt5 and rabbitMQ listener

## Dependencies
- RabbitMQ

## Quiq start

Create venv:
- for Windows:
``` bash
python -m venv venv
.\venv\Scripts\activate
```
- for Linux:
```bash
python3 -m venv venv
source venv/bin/activate
```

1. pip install -r requirements
2. python main.py
